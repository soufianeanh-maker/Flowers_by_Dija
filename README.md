
# Flowers_by_Dija - Simple flower bouquet website

This project is a simple template to showcase and sell flower bouquets. The site content is now in English.

Quick edits:
- Replace product images or links in `shop.html` with your own photos.
- Use the contact page to receive messages from customers; WhatsApp integration can be added later if desired.

Run locally: open `index.html` in a browser (no server needed for basic usage).

Deploy to GitHub Pages (free hosting):
1. Create a GitHub account if you don't have one.
2. Create a new repository on GitHub named something like `flowers-by-dija`.
3. Upload or push all files from this folder to that repository.
   - If you have git installed, use `git init`, `git add .`, `git commit -m "initial"`, and `git branch -M main` then `git remote add origin <repo-url>` and `git push -u origin main`.
   - If you don't have git, you can upload files from the GitHub website using the `Upload files` button.
4. In GitHub repository settings, enable GitHub Pages and choose `main` as the branch.
5. After a few minutes, your site will be available at `https://<username>.github.io/<repo-name>`.

No need to buy hosting — GitHub Pages is free and enough to make the site visible on the web. If Google should find it, wait a few days after publishing and optionally submit your URL to Google Search Console.
